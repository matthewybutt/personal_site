Portfolio

Bugs On A Rug

Bugs On A Rug is a matching game where players try and clear the bugs off of the rug. Play co-op or versus and have some fun!

First project for General Assembly in week 3. A solo project developed in one week.

Technologies Used: HTML5, CSS3, JavaScript

my kid's firsts

This app is a digital baby book for keeping track of all your child's big milestones! Sign up to keep an active record using blog posts and photo uploads of all the firsts your child does.

Second project for General Assembly in week 6. A solo project developed in one week.

Technologies Used: HTML5, CSS3, Ruby, Ruby on Rails, PostgreSQL, Paperclip, Heroku, Amazon Web Services

Pixel Spot

Pixel Spot is a photo location scouting app. Using Flickr's photo hosting and geo-tagging features in conjunction with Google Maps geo-location function users can find the exact spots of photo locations. Search for a spot with built in tag fields or create custom tags.

Third project for General Assembly in week 9. A group project developed in nine days with a team of 4 other developers.

Technologies Used: HTML5, CSS3, JavaScript, Node.js, ExpressJS, MongoDB, jQuery, Google API, Flickr API, GeoNames API, Heroku, MongoLab

Playtime Picks

Playtime Picks is an app for anyone who need to find an activity to do with their kid! Sign-up to add activities to the list or leave comments on an activity.

Final project for General Assembly in week 12. A solo project developed in one week.

Technologies Used: HTML5, CSS3, JavaScript, AngularJS, Node.js, ExpressJS, MongoDB, JSON Web Token (JWT), jQuery, Heroku, MongoLab
